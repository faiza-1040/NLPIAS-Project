# NLP Interview Assessment System (NLPIAS)

## Project Overview

NLPIAS is an AI and NLP-Based Interview Assessment System developed to automate and improve the recruitment process for IT organizations. The system assists recruiters by performing resume parsing, resume-job matching, candidate ranking, AI-based interview generation, and interview evaluation.

Candidates can create professional resumes using an integrated resume builder, upload resumes, apply for jobs, participate in mock interviews, and attend AI-driven interviews using text or voice responses. The system evaluates interview performance using NLP techniques and generates detailed feedback and scores.

For real interviews, the platform also provides basic interview monitoring features such as camera availability detection, multiple-face detection, and attention-related indicators. These indicators are included as supporting information in recruiter reports.

The objective of NLPIAS is to reduce recruitment time, improve hiring accuracy, minimize bias, and provide a smarter recruitment experience for both recruiters and candidates.
